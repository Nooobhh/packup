## [0.3.0] - 2026-07-03 — 生成工作台:自然语言输入 + 选点确认 + 时段制快速排程
### Added
- 一句话自然语言输入:「香港3天2晚 city walk+美食」自动解析目的地/天数/偏好,表单收敛为搜索框 + 链接粘贴区(c91ceef)
- 选点确认页:提取的 POI 先勾选再排程(已验证默认勾选),只有选中的进入行程(313eed7)
- 重排/改交通 API:只重算受影响路段不重跑 LLM,并发冲突时拒绝覆盖并提示刷新(098ae29)
- 地图画高德真实路线(公交分段拼接、缺段直线兜底),相邻 ≤250m 地点聚合为组合节点(29b35c4)
- 生成中断可续:行程 ID 开跑即下发并展示续跑链接,刷新页面不再丢失本次生成(d996cb7)
### Changed
- 行程粒度从分钟时刻表改为上午/下午/晚上时段,每段交通按距离自动推荐(步行/公交/驾车比价)(cc01d29)
- 生成耗时从 12-15 分钟压到机器耗时 ≤5 分钟:超时预算集中管理,段内超时走部分成功不中断(36ed23f)
### Fixed
- 空行程根治:地点名被改写导致整天剔除的问题改为 id 引用,并保证选中地点非空时行程必非空(cc01d29)
- 失败链接的错误信息不再泄漏内部指令与本机路径,统一为一句话摘要(d71eb08)
- 英文目的地 + 偏好(如 Osaka food)不再被整体当作地名导致查证全败(d996cb7)
